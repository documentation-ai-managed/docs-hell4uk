## Overview
Add docs to keep your project organized and up to date.

<Steps>
  <Step title="Open your project">
    ```
    From the dashboard, select your project.
    ```
  </Step>
  <Step title="Click 'Add Doc'">
    ```
    Use the 'Add Doc' button at the top right.
    ```
  </Step>
  <Step title="Fill in details">
    ```
    Add a document title and content using the editor.
    ```
  </Step>
  <Step title="Save and view">
    ```
    Hit 'Save'. Your doc now appears in the sidebar.
    ```
  </Step>
</Steps>
